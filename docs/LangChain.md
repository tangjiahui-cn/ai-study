# LangChain

LangChain 是什么？

LangChain是一个工具，用于快速构建基于LLM的智能代理和应用。
它构建的agents基于LangGraph，提供更多能力例如持久化、流式处理等。
LangChain的目的是让 LLM 入门变得简单。

LangGraph 是什么？

LangGraph 是一个低级编排框架和运行时，用于长时间管理、部署有状态 agent。

## LangChain 理念

- LLM是核心技术
- LLM与外部数据源结合
- 未来应用呈现Agent化
- 构建可靠、可用于生产环境的agent很困难

## LangChain 两个核心：

- 标准化输入输出，让开发者无缝使用最好的模型。
- 通过编排模型使复杂流程变得更容易，更方便定义工具。
